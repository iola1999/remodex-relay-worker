# Remodex Relay Worker

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/iola1999/remodex-relay-worker)

This repository is a Cloudflare Worker deployment adaptation of the Remodex relay.

The original relay code and protocol notes live in the Remodex repository:

https://github.com/Emanuele-web04/remodex/tree/main/relay

This version keeps the relay behavior but adapts the runtime to Cloudflare Workers and Durable Objects.
